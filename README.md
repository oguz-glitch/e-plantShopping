# e-plantShopping# e-plantShopping

A React-based plant shopping application built as part of the IBM Developing Front-End Apps with React course on Coursera.

## Project Name: e-plantShopping

## Features
- Browse plants by category
- Add plants to shopping cart
- Update quantities and remove items
- Calculate total cost

## Technologies Used
- React
- Redux Toolkit
- CSS3
- JavaScript (ES6+)
